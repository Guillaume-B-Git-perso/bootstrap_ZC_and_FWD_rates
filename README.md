# bootstrap_ZC_and_FWD_rates
Bootstrapping of market instruments to get zero-coupon rates' curve and 3/6-months forward rates' curves

This project contains a Python notebook for bootstrapping market instruments to get the zero-coupon rates' curve and the 3/6-months forward rates' curves.

The notebook is interactive and can be executed directly on Google Colab.
